# NGINX

## Installation on Linux
1. Install and update node version  
    `sudo apt-get install nodejs`  
    `sudo npm install -g n`  
    `sudo n stable`  
2. `sudo apt-get install nginx`

## Execuation
1. Start NGINX: `sudo service nginx start`
2. Reload upon changes in config: `service nginx reload`
3. Status: `service nginx status`

## Configuration
1. Config file: `/etc/nginx/nginx.conf`
2. Remove default configs from `sites-*` - `default` files
3. To route to localhost, add server to the config file:
    ```
    server {
        listen       80;
        server_name  json-server;

        location / {
          proxy_pass http://localhost:3000/;
          include /etc/nginx/proxy_params;
        }
    }
    ```
